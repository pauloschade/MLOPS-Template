# About

This is a template repository for a Python Machine Learn project. It is intended to be used as a starting point for a new project.

It creates a pipeline for project deployment.

## Project Structure
The project is organized as follows:

```
.
├── README.md
├── data
├── models
├── notebooks
├── src
│   ├── __init__.py
│   ├── predict.py
│   ├── train.py
│   └── process.py
```

## How to use

```
cookiecutter https://github.com/pauloschade/MLOPS-Template/ --checkout main
```
